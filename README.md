# finance-platform
Full-stack personal finance platform built with Django and React. Features user authentication, budgeting, transaction management, account tracking, and a modular architecture designed to showcase scalable system design and real-world engineering practices.

## Running the App

To run this application locally, follow these steps:

1. **Start the Frontend**
   - Open a terminal window
   - Navigate to the `frontend` directory
   - Run:
     ```bash
     npm run dev
     ```

2. **Start the Backend**
   - Open a second terminal window
   - Navigate to the `backend` directory
   - Run:
     ```bash
     python manage.py runserver
     ```

## Viewing the Application

Once both servers are running, open your browser and visit:

**http://localhost:5173/login**
